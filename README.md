# c-english

A header file that enables writing code using only English sentences.

# example

## Hello, World!

```c
#include "c_english.h"
#include <stdio.h>

An_(integer)_return_type_(main)_function_is_defined_as_follows
    Prints_the_character_("Hello, World!")_to_the_screen

    Returns_the_value_(0);
End_the_block
```
